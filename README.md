# COVID-19 Data Analysis

## Project Overview
This project analyzes a COVID-19 dataset using Python to identify trends and insights through exploratory data analysis (EDA).

## Features
- Data Cleaning
- Missing Value Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure
├── Covid.y.ipynb
├── covid_data.csv
├── images/
└── README.md

## Results
- Analyzed total cases, deaths, recoveries, and active cases.
- Created visualizations to identify patterns and trends.
